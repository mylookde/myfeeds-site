Installation Guide

Get MyFeeds up and running on your WordPress site in under 3 minutes. No developer knowledge required.

MyFeeds Free Plugin Version 1.0.0 · WordPress 5.8+ · PHP 7.4+
Download ZIP
1

Download the plugin

Click the download button above to get the myfeeds.zip file. Save it somewhere you'll find again — your Downloads folder is fine. No need to unzip it.

2

Open your WordPress admin

Log in to your WordPress site. You should land on the Dashboard — the main admin page with the sidebar on the left.

3

Go to Plugins → Add New Plugin

In the left sidebar, hover over Plugins and click Add New Plugin. Then click the Upload Plugin button at the top of the page.

4

Upload the ZIP file

Click Choose File, select the myfeeds.zip you just downloaded, then click Install Now. WordPress will unpack and install the plugin automatically.

Note: If you see an error about file size limits, contact your host to increase the upload limit — or install via FTP by uploading the unzipped folder to /wp-content/plugins/.
5

Activate the plugin

After installation, click Activate Plugin. MyFeeds is now ready to use — you'll see it in your sidebar under MyFeeds.

6

Add your first feed

Click MyFeeds in the sidebar, then Add Feed. Paste your AWIN (or other network) feed URL, give it a name, and let the import run in the background. On the free plan you can add 1 feed and show up to 3 products in a post.

Want more feeds and unlimited products? Upgrade to Pro or Premium from the Manage Plan page inside the plugin — you get a 7-day free trial on all paid plans.
